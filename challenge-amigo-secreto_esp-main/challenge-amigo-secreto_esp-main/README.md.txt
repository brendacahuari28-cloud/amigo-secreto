# Amigo Secreto

Pequeña aplicación para practicar lógica de programación: variables, condicionales, funciones, listas y bucles.

## Funcionalidades
- Agregar participantes (validación: no vacío y no duplicados).
- Eliminar participantes.
- Sortear garantizando que nadie obtenga su propio nombre.
- Mostrar resultado en pantalla.

## Cómo ejecutar (local)
1. Extrae la carpeta.
2. Abre `index.html` en tu navegador **o**
   - Levanta un servidor: `py -m http.server 8000` y abre `http://localhost:8000`.

## Estructura
- `index.html` — interfaz + lógica (o separada en `app.js`).
- `style.css` — estilos.
- `screenshots/` — capturas.
